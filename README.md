# ⚔️ Java OOP Battle Game

A simple 2-player turn-based battle game built with Java to demonstrate core OOP principles.

## 🎮 Features

- 2-player turn-based combat
- Multiple character classes (Warrior, Mage, Archer)
- Unique skills and attack behaviors
- Console-based gameplay

## 🧠 OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## ⚙️ Compile

```bash
javac -d bin src/*.java
```

## ▶️ Run

```bash
java -cp bin Main
```

## 🔁 Workflow

# 1. แก้โค้ดใน src/

# 2. compile ใหม่

```bash
javac -d bin src/*.java
```
